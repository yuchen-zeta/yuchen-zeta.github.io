day02 - 基本DOS命令

发表于2020-08-02更新于2020-08-02字数统计631

Tab + 向上 自动补全 | Ctrl + C 强制终止命令

基本命令

cd.. 返回上级目录
cd\ 返回根目录
d: 进入D盘
cls 清空所有dos操作
dir 显示当前盘符文件信息
rd . /s/q 清空磁盘

浏览文件信息

type 文件名.扩展名  显示
type 文件名.扩展名 | more  分页显示
例如：dir c:\windows | more
(任何命令，只要输出过多都可以用 后缀 | more 分页显示,按空格输出下一页)

创建文件

echo world is so amazing ! 输出到屏幕
echo 字符串 >test.txt 输出到文件，覆盖
echo 字符串 >>c:\test.txt 输出到文件，追加
若文件不存在，则创建该文件

copy con 文件名.扩展名
con可暂时理解为屏幕，即在dos窗口中写内容，然后保存到指定文件
输入命令后回车开始编写 ， Ctrl + Z ，回车结束编写

删除文件

del 文件名.扩展名 删除该文件
del *.txt 删除所有 txt/··· 结尾的文件
del . 删除所有文件
del . /s/q  无提示删除所有文件
/s 所有根目录

文件夹

md 文件名  创建文件夹
attrib +h 文件名  隐藏文件夹
attrib -h 文件名 取消文件夹
attrib +h +s +a 文件名 将文件夹隐藏设为受保护的系统文件夹
dir /a 显示当前磁盘所有文件，包括系统文件

快速生成一个指定大小的空文件，若将其恶意隐藏为系统文件，一般较难发现

fsutil file createnew c:\system.ini 1024000000
在c盘创建1G的空文件

复制和移动

copy 源文件地址 目的地址 复制源文件到目的地址
如：D :\ test \ a > copy hello.txt ..\b\
将test中的a文件夹下的hello.txt 复制到 test 的b文件夹下
D :\ test \ a > copy \test\b\hello.txt .
将b文件夹下的hello.txt 复制到 当前文件夹 a中
move 源文件地址 目的地址 移动源文件到目的地址
ren 旧名 新名 重命名

修改关联

下毒：
assoc .txt=exefile  将.txt文件作为.exe文件打开
修改打开txt文件的程序 使其无法打开
解药：
assoc .txt=txtfile  恢复关联

关机

shutdown -s -t 100 定时关机，单位秒
shutdown -s -f -t 秒 定时强制关机，无任何保存提示
shutdown -s -f -t 秒 -c “别玩儿啦，洗洗睡吧！”
shutdown -r -t 秒 定时重启
shutdown -a 取消一切定时
shutdown -l 注销，同logoff命令相同
